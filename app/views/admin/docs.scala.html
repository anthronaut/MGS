Anleitung

Vorgehen zur Erstellung einer neuen Ausstellung

Zum Erstellen einer neuen Ausstellung können Sie im Ausstellungsmanagement einfach auf den einzigen Button drücken und die Ausstellungsdaten eingeben.

Es ist erforderlich, einen Ausstellungstitel anzugeben.

Sie müssen kein Eröffnungsdatum oder Schliessungsdatum angeben.

Über das Häckchen "Ausstellung veröffentlichen" können Sie festlegen, ob die Ausstellung im Frontend zu sehen ist, oder nicht.

Im Feld "@Messages("comment")" können Sie sich Notizen speichern.

Die Titelbilder der Ausstellung können Sie hinzufügen, nachdem die Ausstellung hinzugefügt wurde, indem Sie im Ausstellungsmanagement auf Ausstellung bearbeiten klicken.

Vorgehen zum Bearbeiten der Ausstellung

Im Ausstellungsmanagement können Sie die Ausstellung, die bearbeitet werden soll ausgewählt werden. Danach haben Sie die Möglichkeit, neue Inhalte hinzuzufügen.

Vorgehen zum Hinzufügen eines Inhaltes

Wenn Sie auf den Button zum Bearbeiten einer Ausstellung geklickt haben, können Sie neue Inhalte hinzufügen, indem Sie unter "@Messages("cat.title")" auf den Button "@Messages("cat.add")" klicken, um einen neuen Inhalt hinzuzufügen.

Dabei kann es sich entweder um einen Menüeintrag, oder gleich um einen spezifischen Inhalt handeln.

Hierbei ist es wichtig, dass das Feld "@Messages("sub")" korrekt ausgefüllt wird, dieses lässt sich nicht mehr verändern.

Alle anderen Werte können Sie später noch bearbeiten, wenn Sie die Inhalte hinzufügen.

Die Verschiedenen Inhaltstypen des Tabletguides

Um einen Inhalt zu bearbeiten, können Sie den Inhalt in einer Ausstellung unter "@Messages("cat.title")" finden, und auf den Button "@Messages("view")" klicken.

Dort haben Sie die Möglichkeiten, die Multimediainhalte hinzuzufügen und die Legenden anzugeben.

Bei allen Inhaltstypen, die kein eigenes Textmodul haben, kann über das Feld "@Messages("comment")" eine Legende eingegeben werden, die auch im Frontend unter dem jeweiligen Inhalt angezeigt wird.

Bild

Mit dem Modul Bildergallerie können Sie eine Gruppe von Bildern hinzufügen, und auch einzelne Legenden zu einem Bild definieren.

Format: jpg

Film

Das Film-Modul dient dazu, Filme zu speichern und weisse Schimmel zu schreiben.

Über "@Messages("file.associated") kann ein Hintergrundbild oder eine Vorschau für den Film festgelegt werden, dies ist aber nicht zwingend Notwendig.

Format: mp4

Ton

Der Tabletguide bietet Inhen die Möglichkeit, Audiodateien abzuspielen, über das Ton-modul.

Format: mp3

Text

Im Textfeld haben Sie die Möglichkeit, einen beliebigen Text einzugeben und zu formatieren. Da das Endformat nicht genau gleich wie der gezeigte Text im Feld ist, empfiehlt es sich, das Resultat über die Vorschau zu kontrollieren.

HTML

Im HTML-Feld kann direkt html code in den Tabletguide eingefügt werden. Dies ist ein Feld mit erweiterter Funktionalität, und sollte nur verwendet werden, wenn es nicht möglich ist, einen anderen Inhaltstypen auszuwählen.
